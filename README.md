# EarthPol Suggestions

This repository stores community suggestions that can be turned into actionable work.

## How suggestions appear here
Suggestions are created as GitHub Issues, typically via a Discord integration.

Each suggestion issue uses this format:

**Suggestion:** <title>

**Description**
<text>

**Author:** <discord username>

## Labels
- `source:discord` - came from Discord
- `type:suggestion` - suggestion intake
- `status:triage` - new, needs review
